O Projeto foi desenvolvido de acordo com as normas do PHP-FIG/PSR´s.

Autoloading via composer. Estou utilizando o projeto "SwiftMailer" para envio de email.

Para rodar o projeto, criar um banco de dados no MySQL e inserir o arquivo usuarios.sql

Alterar o arquivo App/Di/Container.php com os dados de acesso.

É necessário configurar um virtual host apontando para a pasta public do projeto, no apache(necessário mod_rewrite ativado).
